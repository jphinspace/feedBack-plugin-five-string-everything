Chart Retuner
v0.4.3

Remaps every chart onto your real instrument's tuning (bass and guitar profiles, customizable below).

Target Tunings

Every chart is remapped onto a target tuning — any pitches, 4-8 strings. Each arrangement class (bass / rhythm / lead) has its own profile below, and all three pick from the same pool: built-in presets (not editable/deletable) listed first, then your saved custom tunings. Any profile may use any tuning — a bass tuning on a guitar arrangement is fine. Add your own for AEADG, BbEbAbDbGb, a 7-string, or anything else. Strings can only be added/removed from the ends, never the middle — a new string gets a fresh default note + color; existing strings' colors never shift, and a removed string's color is never remembered for later.

Apply to the highway immediately — no reload required.

Highest fret this tuning remaps onto. Most charts never go past fret 20 — raise it for extended-range instruments or the occasional high transcribed solo, lower it for a genuinely short-necked instrument.

Clamp a capo on top of the string pitches below — one fret = one half-step up per string, and the frets above (max fret − capo) fall off the end of the neck. 0 = no capo. Saved with the tuning (for a capo you never take off); the player-screen sliders can still adjust it later.

Shift every chart up/down whole octaves before remapping — no key change involved. +1 plays an E-standard bass chart on guitar strings note-for-note; save +1 on a cello tuning to play bass charts an octave up by default.
